## 定义

已知 x = [An, A(n-1), ... A1, A0]
则 x 的十进制值为： x = 2^n * An + 2^(n-1) * A(n-1) + ... + 2^1 * A1 + 2^0 * A0

## 十进制转二进制 

可得 A0 = x % 2

=>

可知 (x - 2^0 * A0) / 2 = 2^(n-1) * An + 2^(n-2) * A(n-1) + ... + 2^0 * A1 
可得 A1 = (x - 2^0 * A0) % 2

## 十六进制

每一位的表示：0~9 + ABCDEF

例子:

> deadbeef
> 1111AAAA
